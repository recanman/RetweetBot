# RetweetBot

RetweetBot is a simple one-script program that will retweet recent tweets from a list of accounts.

## Installation

First, clone the repo (or download it)
Then, use [npm](https://www.npmjs.com/) to install the dependencies.
```bash
cd RetweetBot/src
npm i
```

## Usage

```bash
npm run
```
